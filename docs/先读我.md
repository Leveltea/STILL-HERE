# AI 项目 - Save My Work

这个文件夹包含当前 demo 的可运行版本和源码。

## 直接运行

双击：

`打开保存工作现场.command`

如果 macOS 提示安全限制，可以右键 `保存工作现场.app`，选择“打开”。

## 关闭应用

双击：

`关闭保存工作现场.command`

## 文件夹结构

- `保存工作现场.app`：当前已经打包好的 macOS 应用，可直接打开。
- `项目源码/`：Electron + React + Vite 源码，给 Codex 或开发者继续修改。
- `说明文档/`：迁移、运行、产品上下文说明。

## 当前 demo 重点

这个 demo 不是普通窗口恢复器，而是一个桌面上的“工作记忆胶囊”：

- 用户被打断时，保存当前窗口和思路。
- 自动识别窗口角色，比如沟通、资料、当前产物、本地素材。
- 生成工作断点：当前任务、卡点、回来第一步。
- 恢复时帮助用户接上思路。

当前实现仍是 demo：窗口内容理解主要靠标题、缩略图和模拟规则，不是真正读取聊天/Notion 内容。
